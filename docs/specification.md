# Specification references

The development baseline is the 2026-07-16 ECHOLOOP: PLAYLIST RAID 1.0.0
attachment set referenced by the project history.

Fixed product constraints are the product name, Godot 4.7.1 Standard, GDScript,
Windows 11 priority, Python 3.11, F/J DUO as the default and D/F/J/K CLASSIC as
an option, absolute-time synchronization, successful-input Echo, MISS
Corruption, offline-first behavior, and no external AI API at normal startup.

BeatMap is the only musical time conversion boundary. Schema v1 fixtures remain
supported and generated local charts use schema v2.
